## 第二層：原因診斷

針對每個 gap 關鍵字，判斷問題類型：

- **標題問題**：排名不差但沒人點 → 改標題或 Meta Description
- **內容缺口**：有曝光但沒有對應文章 → 新寫一篇
- **排名太低**：排名 > 20 → 先強化現有文章的 SEO

3. 哪篇需要補充內容以提升排名
